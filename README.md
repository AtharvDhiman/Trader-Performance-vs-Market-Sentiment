# Trader Performance vs Market Sentiment

Project scaffold for analyzing trader performance alongside market sentiment signals.

## Folders

- `data/` - Source CSV files such as historical market data and fear/greed index data.
- `notebooks/` - Jupyter notebooks for exploration and analysis.
- `charts/` - Generated visualizations.
- `report/` - Final writeups, summaries, and exported reports.

## Files

- `requirements.txt` - Python dependencies for analysis notebooks and scripts.
- `.gitignore` - Ignored files and generated artifacts.
